# Requirement

* Flutter 3.7.6
* Dart 2.19.3
* DevTools 2.20.1
* pod 1.12.0


# Installation

```bash
brew install cocoapods
pod setup
```

# Usage

```bash
git clone https://github.com/takuowake/ivent-reception-app.git
flutter run -d chrome
```
